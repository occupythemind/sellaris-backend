## Guest User

### Guest User Clean Up Policy
To avoid huge data bugs, we usually clean up guest user data that meets/exceeds this interval.
Cart → 14 days (from last update)
Wishlist → 30 days (from last update)
Pending Orders → 24–48 hrs (from last update)
Completed Orders → never delete
Payment Records -> Nevr delete

### Ownership Transfer (on login)
Wishlist → always transfer
Cart → merge
Orders → only PENDING
Payments → never
